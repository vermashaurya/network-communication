# Demo Recording Commands

Use these commands while recording terminal demos:

```bash
python projects/udp-chat/server.py
python projects/udp-chat/client.py
```

```bash
python projects/udp-echo/server.py
python projects/udp-echo/client.py
```

```bash
python projects/tcp-even-parity/server.py
python projects/tcp-even-parity/client.py --data 101010 --append-parity-bit
```

```bash
python projects/tcp-crc/server.py
python projects/tcp-crc/client.py --message "networking"
```

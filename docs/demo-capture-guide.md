# Demo Capture Guide

Use this guide to create presentable media for the repository.

## Terminal Demo Checklist

- Record one short run for each socket project.
- Show server start, client request, and server response.
- Keep each clip under 30 seconds.

## Recommended Naming

- `assets/media/udp-chat.gif`
- `assets/media/udp-echo.gif`
- `assets/media/tcp-even-parity.gif`
- `assets/media/tcp-crc.gif`

## Placement

- Reference these files from each project README and the root README.
